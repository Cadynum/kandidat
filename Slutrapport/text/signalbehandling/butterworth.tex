\section{Butterworth filter}
\label{sec:butter}
För att filtera signaler används ett \emph{Butterworth-filter} av första ordningen. Filtret är ett lågpassfilter med minimal fasförskjutning för kausala lågpassfilter.
\begin{figure}[htb]
fig: bodeplot
\end{figure}

\begin{equation}
\label{eq:butter_s}
G(s) = \frac{\omega_0}{\omega_0 + s}
\end{equation}
Överföringsfunktionen i laplace-domän överförs till Z-domän med en samplingsfrekvens på $f_s$ för att kunna implementera filtret på diskreta mätvärden:
\begin{equation}
\label{eq:butter_z}
G(z) = \frac{\omega_0 (z+1)}{2 f_s z-2 f_s + \omega_0 z + \omega_0}
\end{equation}
Genom att sätta $\omega_0 = 2\pi f_0$ och inverstransformera får vi följande differensekvation som kan användas för att i realtid implementera den filtrerade signalen digitalt:
\begin{equation}
\label{eq:butterdiff}
    y[n] = \frac{f_0(x[n]+x[n-1]) - y[n-1](f_0 - 2f_s)}
                {f_0 + 2f_s}
\end{equation}
